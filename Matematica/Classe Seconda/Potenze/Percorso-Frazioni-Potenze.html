La potenza di una frazione

Clicca su ogni elemento per scoprirne il contenuto.

Definizione di potenza

Elevare una frazione a potenza significa moltiplicare la frazione per se stessa tante volte quante indica l'esponente.

In pratica, si elevano sia il numeratore sia il denominatore allo stesso esponente.

\left(\dfrac{a}{b}\right)^{n} = \dfrac{a^{n}}{b^{n}}

La base è la frazione. L'esponente è il numero che indica quante volte moltiplicare. Il risultato è la potenza.

Esempi svolti
Esempio 1
\left(\dfrac{2}{3}\right)^{3} = \dfrac{2^3}{3^3} = \dfrac{2 \times 2 \times 2}{3 \times 3 \times 3} = \dfrac{8}{27}
Esempio 2
\left(\dfrac{5}{4}\right)^{2} = \dfrac{5^2}{4^2} = \dfrac{25}{16}
Esempio 3
\left(\dfrac{1}{6}\right)^{2} = \dfrac{1^2}{6^2} = \dfrac{1}{36}
Caso particolare: esponente 0

Qualsiasi frazione diversa da zero, elevata alla zero, dà sempre 1.

\left(\dfrac{a}{b}\right)^{0} = 1
Esempi
\left(\dfrac{3}{7}\right)^{0} = 1    \left(\dfrac{11}{2}\right)^{0} = 1
Caso particolare: esponente 1

Qualsiasi frazione elevata alla prima dà la frazione stessa.

\left(\dfrac{a}{b}\right)^{1} = \dfrac{a}{b}
Esempi
\left(\dfrac{5}{9}\right)^{1} = \dfrac{5}{9}    \left(\dfrac{2}{3}\right)^{1} = \dfrac{2}{3}
⚠ Errore comune da evitare

L'esponente si applica a tutta la frazione, non solo al numeratore!

✔ Corretto
\left(\dfrac{2}{3}\right)^{2} = \dfrac{4}{9}
✘ Errato:   \dfrac{4}{3}   — l'esponente è stato applicato solo al numeratore!
Le proprietà delle potenze

Clicca su ogni proprietà per studiarla con la regola e gli esempi.

1. Prodotto di potenze con la stessa base

Si mantiene la stessa base e si sommano gli esponenti.

\left(\dfrac{a}{b}\right)^{m} \times \left(\dfrac{a}{b}\right)^{n} = \left(\dfrac{a}{b}\right)^{m+n}
Esempi

\left(\dfrac{2}{5}\right)^{3} \times \left(\dfrac{2}{5}\right)^{2} = \left(\dfrac{2}{5}\right)^{3+2} = \left(\dfrac{2}{5}\right)^{5}

\left(\dfrac{1}{3}\right)^{4} \times \left(\dfrac{1}{3}\right)^{1} = \left(\dfrac{1}{3}\right)^{5} = \dfrac{1}{243}

2. Quoziente di potenze con la stessa base

Si mantiene la stessa base e si sottraggono gli esponenti.

\left(\dfrac{a}{b}\right)^{m} : \left(\dfrac{a}{b}\right)^{n} = \left(\dfrac{a}{b}\right)^{m-n}   con m \geq n
Esempi

\left(\dfrac{3}{4}\right)^{5} : \left(\dfrac{3}{4}\right)^{3} = \left(\dfrac{3}{4}\right)^{2} = \dfrac{9}{16}

\left(\dfrac{2}{7}\right)^{6} : \left(\dfrac{2}{7}\right)^{4} = \left(\dfrac{2}{7}\right)^{2} = \dfrac{4}{49}

3. Potenza di potenza

Si mantiene la stessa base e si moltiplicano gli esponenti.

\left[\left(\dfrac{a}{b}\right)^{m}\right]^{n} = \left(\dfrac{a}{b}\right)^{m \times n}
Esempi

\left[\left(\dfrac{1}{2}\right)^{3}\right]^{2} = \left(\dfrac{1}{2}\right)^{6} = \dfrac{1}{64}

\left[\left(\dfrac{3}{5}\right)^{2}\right]^{3} = \left(\dfrac{3}{5}\right)^{6}

4. Potenza di un prodotto

La potenza di un prodotto è uguale al prodotto delle potenze dei singoli fattori.

\left(\dfrac{a}{b} \times \dfrac{c}{d}\right)^{n} = \left(\dfrac{a}{b}\right)^{n} \times \left(\dfrac{c}{d}\right)^{n}
Esempio

\left(\dfrac{2}{3} \times \dfrac{3}{4}\right)^{2} = \left(\dfrac{1}{2}\right)^{2} = \dfrac{1}{4}

5. Potenza di un quoziente

La potenza di un quoziente è uguale al quoziente delle potenze.

\left(\dfrac{a}{b} : \dfrac{c}{d}\right)^{n} = \left(\dfrac{a}{b}\right)^{n} : \left(\dfrac{c}{d}\right)^{n}
Esempio

\left(\dfrac{3}{4} : \dfrac{3}{8}\right)^{3} = 2^{3} = 8

🧮 Quando si possono applicare le proprietà?

Le proprietà del prodotto, del quoziente e della potenza di potenza valgono solo quando la base è la stessa.

✔ Si può applicare
\left(\dfrac{2}{3}\right)^{2} \times \left(\dfrac{2}{3}\right)^{3} = \left(\dfrac{2}{3}\right)^{5}

Le basi sono uguali → sommiamo gli esponenti.

✘ Non si può applicare:
\left(\dfrac{2}{3}\right)^{2} \times \left(\dfrac{3}{4}\right)^{3}

Le basi sono diverse → non si possono sommare gli esponenti!

⚠ Sommare o moltiplicare gli esponenti?

È l'errore più frequente. Ecco come distinguere:

Prodotto di potenze → gli esponenti si sommano

\left(\dfrac{a}{b}\right)^{m} \times \left(\dfrac{a}{b}\right)^{n} = \left(\dfrac{a}{b}\right)^{m+n}

Potenza di potenza → gli esponenti si moltiplicano

\left[\left(\dfrac{a}{b}\right)^{m}\right]^{n} = \left(\dfrac{a}{b}\right)^{m \times n}

Trucco: nel prodotto ci sono due basi scritte e si sommano; nella potenza di potenza c'è una base sola con due esponenti e si moltiplicano.

★ I casi particolari: esponente 0 e 1

Esponente 0: qualsiasi numero o frazione (diversi da zero) elevati a 0 danno 1.

\left(\dfrac{7}{11}\right)^{0} = 1   5^{0} = 1   1000^{0} = 1

Esponente 1: qualsiasi numero o frazione elevati a 1 danno se stessi.

\left(\dfrac{3}{8}\right)^{1} = \dfrac{3}{8}   12^{1} = 12

Attenzione: 0^{0} è una forma indeterminata — non ha un valore definito.
📐 Riepilogo delle 5 proprietà

1.   \left(\frac{a}{b}\right)^{m} \times \left(\frac{a}{b}\right)^{n} = \left(\frac{a}{b}\right)^{m+n}

2.   \left(\frac{a}{b}\right)^{m} : \left(\frac{a}{b}\right)^{n} = \left(\frac{a}{b}\right)^{m-n}

3.   \left[\left(\frac{a}{b}\right)^{m}\right]^{n} = \left(\frac{a}{b}\right)^{m \times n}

4.   \left(\frac{a}{b} \times \frac{c}{d}\right)^{n} = \left(\frac{a}{b}\right)^{n} \times \left(\frac{c}{d}\right)^{n}

5.   \left(\frac{a}{b} : \frac{c}{d}\right)^{n} = \left(\frac{a}{b}\right)^{n} : \left(\frac{c}{d}\right)^{n}
